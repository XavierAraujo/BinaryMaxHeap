Folders:

bin: Contains the binary file that resulted from the build procedure

doc: Contains the documentation of the project that was automatically generated from the source files using Doxygen. The documentation can be accesses through doc/index.htm. The documentation can be accesses through doc/html/index.html

test: Contains the unit tests of the project. These unit tests were made using CUnit.

src: Contains the source files (*.c)

inc: Contains the header files (*.h)
